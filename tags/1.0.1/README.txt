RabbitCT (version 1.0)
======================

RabbitCT provides an open platform for worldwide performance-comparison of
backprojection implementations on different architectures using one specific,
clinical, high resolution C-arm CT dataset of a rabbit. This includes a
sophisticated benchmark interface, a prototype implementation in C++, and
image quality measures.

For more information about RabbitCT, please visit the RabbitCT website at this
point: http://www.rabbitct.com/

RabbitCT is a project by the Pattern Recognition Lab at the Universität
Erlangen-Nürnberg. http://www5.cs.fau.de/
